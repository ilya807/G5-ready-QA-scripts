#  Games QA Automation Framework (Python)

Тестовый проект по автоматизации игровых механик (Inventory, Economy) для симуляции реального рабочего процесса в геймдеве.

## Используемый стек:
*   **Язык:** Python 3.10+
*   **Фреймворк:** Pytest
*   **Управление данными:** JSON-driven testing
*   **Окружение:** Git, GitHub Actions (в планах)

## Тестовые сценарии:
*   **`Economy Tests`:** Проверка баланса внутриигрового магазина, валидация цен, логика покупки и списания валюты.
*   **`Inventory System`:** Тестирование лимитов слотов, добавление и удаление предметов, проверка переполнения инвентаря.

## Как запустить тесты локально:

1.  **Клонировать репозиторий:** `git clone github.com`
2.  **Перейти в папку проекта:** `cd G5-ready-QA-scripts`
3.  **Установить зависимости:** `pip install -r requirements.txt` (или `python3 -m pip install pytest --break-system-packages` на Mac)
4.  **Запустить тесты:** `python3 -m pytest`
